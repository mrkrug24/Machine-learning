Идея решения:
1. Предобработка текста:
    - удалить скобки и их содержимое
    - оставить только буквы и цифры
    - преобразовать буквы в нижний регистр
    - преобразовать слова в начальную форму
2. Рассматриваем все непрерывные блоки из строк размера от 2 до min(10, количества всех строк / 4) (величины выбраны экспериментальным путем)
3. Сравниваем блоки одинаковых размеров через Levenshtein
4. Заполняем матрицу, где в [i][i] лежит 0 или 1, в зависимости от значения Levenshtein для блоков i и j
    - если Levenshtein <= 0.075, то пишем 1 (величина выбрана экспериментальным путем)
    - иначе пишем 0
5. Нам нужна строка с наибольшим количеством 1, если таких несколько, то выбираем ту, которая соответсвует наибольшему размеру блока
6. В процессе заполнения матриц запоминаем интересующие нас величины
7. После вычислений по лучшей строке восстанавливаем множестов интервалов строк, относящихся к припеву
